About SatSeg

SatSeg is an AI-powered satellite image analysis system that performs pixel-level semantic segmentation using a custom U-Net architecture, classifying land into 6 environmental categories and performing automated vulnerability risk assessment.

🧠

U-Net Architecture

Encoder–decoder with skip connections for pixel-level accuracy. Trained on 128×128 satellite patches with weighted categorical cross-entropy loss to handle class imbalance.

🎨

RGB Mask Encoding

Ground-truth masks use distinct RGB colours per class. A threshold-based decoder converts colour → integer class labels (0–5) during preprocessing.

⚖️

Weighted Loss

Class weights [0.5, 2.0, 1.0, 0.7, 1.2, 1.5] emphasise rare classes (Water, Urban) to counter the dominant vegetation bias in the dataset.

⚡

FastAPI Backend

RESTful API with 5 endpoints. Accepts satellite image uploads, runs TensorFlow inference, computes vulnerability metrics, and returns a Base64 PNG segmentation mask.

Vulnerability Methodology

🏜️ Drought Vulnerability

Calculated by correlating Dry Land percentage against Total Vegetation. A high drought risk (Dry > 50%, Veg < 30%) indicates a critical lack of soil moisture and vegetation health, leading to potential land degradation and loss of agricultural productivity.

🌊 Flood Vulnerability

Determined by Water Body surface area in relation to Natural Absorption (vegetation cover). Areas with over 40% water coverage and low vegetation are flagged as high risk, as the lack of green cover significantly reduces the land's ability to absorb excess moisture during heavy rainfall.

🌱 Ecosystem Resilience

Assessed via Total Biomass (Dense + Light Vegetation). A healthy ecosystem (>70% cover) provides essential services like carbon sequestration, local climate regulation, and soil stabilization. Coverage below 40% is considered poor, indicating a fragmented and vulnerable habitat.

🏙️ Urban Environmental Stress

Measured by Impervious Surface (Urban) density. High urban concentration (>30%) causes "Heat Island" effects and disrupts natural water cycles. Monitoring urban stress is vital for sustainable city planning and temperature mitigation strategies.

🛡️

5-Module Risk Engine

Drought, Flood, Ecosystem Health, Urban Stress — each independently scored with thresholds, combined into a final environmental vulnerability status.

📊

Pixel-Level Analysis

Every pixel in the 128×128 output is assigned a class. Coverage percentages are computed and fed directly into the vulnerability rule engine.

Processing Pipeline

1

Image Upload

Satellite image (JPG/PNG) uploaded via browser → validated and sent to the FastAPI backend.

2

Preprocessing

Decoded as BGR, resized to 128×128, normalised: (img − mean) / (std + 1e-7)

3

U-Net Inference

Model outputs (128, 128, 6) softmax probabilities. argmax across axis-2 → class map (0–5).

4

Colour Mapping

Class map coloured by palette → resized to 800×800 (INTER_NEAREST) → PNG → Base64.

5

Vulnerability Analysis

Pixel percentages calculated per class → fed into 5-module rule engine → final risk status returned.
